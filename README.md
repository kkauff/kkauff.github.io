# kkauff.github.io

Personal website built with Next.js, TypeScript, and Tailwind CSS. Automatically deployed to GitHub Pages.

## Development

```bash
# Install dependencies
npm install

# Run development server
npm run dev

# Build for production
npm run build
```

Visit [https://kkauff.github.io](https://kkauff.github.io) to see the live site.

## Tech Stack

- **Next.js 14** - React framework with App Router
- **TypeScript** - Type-safe JavaScript
- **Tailwind CSS** - Utility-first CSS framework
- **GitHub Actions** - Automated deployment

## Deployment

The site automatically deploys to GitHub Pages when changes are pushed to the `master` branch.
